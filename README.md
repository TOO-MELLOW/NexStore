# 🛍️ NexStore — E-Commerce Platform

A full e-commerce store built with React.

## 📁 Project Structure

```
nexstore/
├── index.html          ← The main HTML page
├── package.json        ← Project dependencies
├── vite.config.js      ← Build configuration
└── src/
    ├── main.jsx        ← Entry point (don't edit this)
    └── App.jsx         ← YOUR ENTIRE STORE IS HERE
```

## 🚀 How to Run (StackBlitz)

1. Drag this entire folder into stackblitz.com
2. StackBlitz will install everything automatically
3. Your store preview appears on the right side
4. Click "Deploy" to put it live on Vercel

## 🔑 Demo Login

- **Regular user:** any email + any password
- **Admin access:** admin@test.com + any password

All your products are in `src/App.jsx`.
